HTML


What is HTML?

1.HTML stands for HyperText Markup Language.
2.HTML is also the standard language which is used to create web pages.
3.It tells the web browser what to show and how it should be structured.


What is HTML used for?

1.HTML is for creating Web pages.
2.Displaying text, images, videos, and links on the web.
3. Structuring content with headings, paragraphs, lists, and tables.
4. Building the overall layout and design of web pages.
5.Embedding multimedia elements like audio and video.

How HTML usally works?

1. HTML uses tags to mark up content.


2. Tags define the structure and meaning of elements on a webpage.


Tags are written inside angle brackets <>


4.Most tags have an opening tag and a closing tag.


Basic structure of HTML

1. The basic structure of an HTML document always includes the DOCTYPE declaration, html, head, and body elements.


Why is HTML important?

1.Some people say that HTML is the backbone of the web pages because it provides the basic structure and content to web pages which are displayed to users.


2. HTML works perfectly together with CSS and JavaScript to create complete web experiences.


Fun facts about HTML

1. The first version of HTML was created by Tim Berners-Lee in 1991.


2. The first web page was also created by Tim Berners-Lee and it is still available online.


3. The latest version of HTML is HTML5, which was released in 2014.
